# 🧠 LearnGenie AI – Adaptive Generative AI Learning Platform

LearnGenie AI is an **adaptive AI-powered learning platform** that generates personalized study materials, quizzes, and interactive mind maps using **Generative AI and machine learning techniques**.

The platform analyzes the learner’s topic input and automatically produces structured learning content, evaluates understanding through quizzes, and visualizes knowledge using mind maps.

The goal of the project is to create an **intelligent learning assistant** that helps students understand complex topics faster through **AI-generated explanations, adaptive learning paths, and visual knowledge representation**.# 🧠 LearnGenie AI – Adaptive Generative AI Learning Platform

LearnGenie AI is an **adaptive AI-powered learning platform** that generates personalized study materials, quizzes, and interactive mind maps using **Generative AI and machine learning techniques**.

The platform analyzes the learner’s topic input and automatically produces structured learning content, evaluates understanding through quizzes, and visualizes knowledge using mind maps.

The goal of the project is to create an **intelligent learning assistant** that helps students understand complex topics faster through **AI-generated explanations, adaptive learning paths, and visual knowledge representation**.

---

# 🚀 Key Features

• **AI-Generated Study Material** – Generates structured learning content for any topic
• **Interactive Mind Maps** – Visual representation of concepts for better understanding
• **AI Chat Assistant** – Conversational AI for topic clarification and learning support
• **Auto Quiz Generation** – Automatically generates quiz questions for knowledge evaluation
• **Adaptive Learning Engine** – Recommends next topics based on learner progress
• **PDF Export** – Download generated study material as a PDF
• **User Tracking** – Stores user data and learning activity
• **Modern UI** – Built with Streamlit and custom CSS styling

# ⚙ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/22AD040/learngenie-ai.git
```

### 2️⃣ Navigate to the project

```
cd learngenie-ai
```

### 3️⃣ Create virtual environment

```
python -m venv venv
```

### 4️⃣ Activate virtual environment

Windows

```
venv\Scripts\activate
```

Mac / Linux

```
source venv/bin/activate
```

### 5️⃣ Install dependencies

```
pip install -r requirements.txt
```

---

# ▶ Running the Application

Run the Streamlit application:

```
streamlit run app.py
```

The application will start at:

```
http://localhost:8501
```

# 📊 Workflow of the System

1️⃣ User enters a **learning topic**
2️⃣ AI generates **structured study material**
3️⃣ Key sections are extracted to create a **mind map**
4️⃣ The system generates **quiz questions** to evaluate understanding
5️⃣ AI chat assistant helps clarify concepts
6️⃣ Study material can be **downloaded as a PDF**
